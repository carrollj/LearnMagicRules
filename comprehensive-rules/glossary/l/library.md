[Back to L entries](README.md)

Source: Magic: The Gathering Comprehensive Rules glossary (effective June 19, 2026)

## Definition

1. A zone. A player’s library is where that player draws cards from.
2. All the cards in a player’s library.
See rule [401](../../4-zones/401-library.md#rule-401), “Library.”
