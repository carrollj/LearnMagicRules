## Definition

A Lander token is a colorless artifact token with “{2}, {T}, Sacrifice this token: Search your library for a basic land card, put it onto the battlefield tapped, then shuffle.” For more information about predefined tokens, see rule [111.10](../../1-game-concepts/111-tokens.md#rule-111-10).
