## Definition

There are several ways to lose the game. See rule [104](../../1-game-concepts/104-ending-the-game.md#rule-104), “Ending the Game,” rule [810.8](../../8-multiplayer-rules/810-two-headed-giant-variant.md#rule-810-8) (for additional rules for Two-Headed Giant games), rule [809.5](../../8-multiplayer-rules/809-emperor-variant.md#rule-809-5) (for additional rules for Emperor games), and rule [903.10](../../9-casual-variants/903-commander.md#rule-903-10) (for an additional rule for Commander games).
