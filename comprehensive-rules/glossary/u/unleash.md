# Unleash

Source: Magic: The Gathering Comprehensive Rules glossary (effective June 19, 2026)

## Definition

A keyword ability that allows a creature to enter the battlefield with a +1/+1 counter on it and stops it from blocking if it has a +1/+1 counter on it. See rule [702.98](../../7-additional-rules/702-keyword-abilities.md#rule-702-98), “Unleash.”
