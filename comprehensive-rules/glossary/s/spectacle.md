Source: Magic: The Gathering Comprehensive Rules glossary (effective June 19, 2026)

## Definition

A keyword ability that allows certain spells to be cast for an alternative cost if an opponent has lost life. See rule [702.137](../../7-additional-rules/702-keyword-abilities.md#rule-702-137), “Spectacle.”
