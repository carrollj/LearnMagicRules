Source: Magic: The Gathering Comprehensive Rules glossary (effective June 19, 2026)

## Definition

The number of cards a player draws as a game begins. In most games, each player’s starting hand size is seven. See rule [103.5](../../1-game-concepts/103-starting-the-game.md#rule-103-5).
