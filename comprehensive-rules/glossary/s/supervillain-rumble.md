## Definition

A Free-for-All game in which each player is an archenemy. See rule [806](../../8-multiplayer-rules/806-free-for-all-variant.md#rule-806), “Free-for-All,” and rule [904](../../9-casual-variants/904-archenemy.md#rule-904), “Archenemy.”
