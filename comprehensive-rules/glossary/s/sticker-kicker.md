# Sticker Kicker

[Back to glossary index](../README.md)
[Back to S entries](README.md)

Source: Magic: The Gathering Comprehensive Rules glossary (effective June 19, 2026)

## Definition

A kicker variant that can add a sticker to the spell with the ability and give a player a ticket counter. See rule [702.33h](../../7-additional-rules/702-keyword-abilities.md#rule-702-33h).
