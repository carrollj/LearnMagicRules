Source: Magic: The Gathering Comprehensive Rules glossary (effective June 19, 2026)

## Definition

A triggered ability that triggers when a game state is true rather than triggering when an event occurs. See rule [603.8](../../6-spells-abilities-and-effects/603-handling-triggered-abilities.md#rule-603-8).
