# Set in Motion

[Back to glossary index](../README.md)
[Back to S entries](README.md)

Source: Magic: The Gathering Comprehensive Rules glossary (effective June 19, 2026)

## Definition

To move a scheme card off the top of your scheme deck and turn it face up. See rule [701.32](../../7-additional-rules/701-keyword-actions.md#rule-701-32), “Set in Motion.”
