## Definition

A subtype that’s correlated to the instant card type and the sorcery card type. See rule [304](../../3-card-types/304-instants.md#rule-304), “Instants,” and rule [307](../../3-card-types/307-sorceries.md#rule-307), “Sorceries.” See rule [205.3k](../../2-parts-of-a-card/205-type-line.md#rule-205-3k) for the list of spell types.
