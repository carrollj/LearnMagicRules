## Definition

A subtype that’s correlated to the artifact card type. See rule [301](../../3-card-types/301-artifacts.md#rule-301), “Artifacts.” See rule [205.3g](../../2-parts-of-a-card/205-type-line.md#rule-205-3g) for the list of artifact types.
