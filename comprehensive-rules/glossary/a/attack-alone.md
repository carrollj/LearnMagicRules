## Definition

A creature “attacks alone” if it’s the only creature declared as an attacker during the declare attackers step. A creature “is attacking alone” if it’s attacking but no other creatures are. See rule [506.5](../../5-turn-structure/506-combat-phase.md#rule-506-5).
