# Attach

Source: Magic: The Gathering Comprehensive Rules glossary (effective June 19, 2026)

## Definition

To take an Aura, Equipment, or Fortification from where it currently is and put it onto a specified object or player. See rule [701.3](../../7-additional-rules/701-keyword-actions.md#rule-701-3), “Attach.”
