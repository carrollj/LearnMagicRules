Source: Magic: The Gathering Comprehensive Rules glossary (effective June 19, 2026)

## Definition

A different name used on promotional or alternate-art versions of some cards. This name has no rules meaning. See rule [201.6](../../2-parts-of-a-card/201-name.md#rule-201-6).
