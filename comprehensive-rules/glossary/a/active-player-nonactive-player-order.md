Source: Magic: The Gathering Comprehensive Rules glossary (effective June 19, 2026)

## Definition

A system that determines the order by which players make choices if multiple players are instructed to make choices at the same time. See rule [101.4](../../1-game-concepts/101-the-magic-golden-rules.md#rule-101-4). This rule is modified for games using the shared team turns option; see rule [805.6](../../8-multiplayer-rules/805-shared-team-turns-option.md#rule-805-6).
