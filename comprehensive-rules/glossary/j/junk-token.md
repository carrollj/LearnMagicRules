## Definition

A Junk token is a colorless artifact token with {T}, Sacrifice this token: Exile the top card of your library. You may play that card this turn. Activate only as a sorcery.” For more information about predefined tokens, see rule [111.10](../../1-game-concepts/111-tokens.md#rule-111-10).
