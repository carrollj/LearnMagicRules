Source: Magic: The Gathering Comprehensive Rules glossary (effective June 19, 2026)

## Definition

1. A card is “face up” if it’s physically positioned so the card front is showing. Cards in some zones are normally kept face up. See section [4](../../4-zones/README.md), “Zones.”
2. A default status a permanent may have. See rule [110.5](../../1-game-concepts/110-permanents.md#rule-110-5) and rule [702.37](../../7-additional-rules/702-keyword-abilities.md#rule-702-37), “Morph.”
