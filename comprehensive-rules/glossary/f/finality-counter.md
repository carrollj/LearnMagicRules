Source: Magic: The Gathering Comprehensive Rules glossary (effective June 19, 2026)

## Definition

A counter that exiles the permanent it is on if that permanent would go to the graveyard from the battlefield. See rule [122.1h](../../1-game-concepts/122-counters.md#rule-122-1h).
