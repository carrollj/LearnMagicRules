Source: Magic: The Gathering Comprehensive Rules glossary (effective June 19, 2026)

## Definition

To choose a set of creatures that will block, declare which attacking creature each creature is blocking, and pay any costs required to allow those creatures to block. See rule [509.1](../../5-turn-structure/509-declare-blockers-step.md#rule-509-1).
