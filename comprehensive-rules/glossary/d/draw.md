# Draw

[Back to D entries](README.md)

Source: Magic: The Gathering Comprehensive Rules glossary (effective June 19, 2026)

## Definition

1. To put the top card of a player’s library into their hand as a turn-based action or as the result of an effect that uses the word “draw.” See rule [121](../../1-game-concepts/121-drawing-a-card.md#rule-121), “Drawing a Card.”
2. The result of a game in which neither player wins or loses. See rule [104.4](../../1-game-concepts/104-ending-the-game.md#rule-104-4).
