# Vanguard

[Back to glossary index](../README.md)
[Back to V entries](README.md)

Source: Magic: The Gathering Comprehensive Rules glossary (effective June 19, 2026)

## Definition

1. A casual variant in which each player plays the role of a famous character. See rule [902](../../9-casual-variants/902-vanguard.md#rule-902), “Vanguard.”
2. A card type seen only on nontraditional Magic cards in the Vanguard casual variant. A vanguard card is not a permanent. See rule [313](../../3-card-types/313-vanguards.md#rule-313), “Vanguards.”
