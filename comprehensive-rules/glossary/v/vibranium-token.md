## Definition

A Vibranium token is a colorless Vibranium artifact token with indestructible and “{T}: Add {C}. This mana can’t be spent to cast a nonartifact spell.” For more information about predefined tokens, see rule [111.10](../../1-game-concepts/111-tokens.md#rule-111-10).
