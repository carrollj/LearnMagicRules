Source: Magic: The Gathering Comprehensive Rules glossary (effective June 19, 2026)

## Definition

A turn created by an effect of a spell or ability. See rule [500.7](../../5-turn-structure/500-general.md#rule-500-7). For rules about extra turns in a multiplayer game using the shared team turns option, see 805.8. For rules about extra turns in a Grand Melee game, see rule [807.4](../../8-multiplayer-rules/807-grand-melee-variant.md#rule-807-4).
