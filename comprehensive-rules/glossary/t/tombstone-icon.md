# Tombstone Icon

[Back to glossary index](../README.md)
[Back to T entries](README.md)

Source: Magic: The Gathering Comprehensive Rules glossary (effective June 19, 2026)

## Definition

An icon that appears in the upper left of some Odyssey block cards that has no effect on game play. See rule [107.9](../../1-game-concepts/107-numbers-and-symbols.md#rule-107-9).
