# Trigger Event

[Back to glossary index](../README.md)
[Back to T entries](README.md)

Source: Magic: The Gathering Comprehensive Rules glossary (effective June 19, 2026)

## Definition

The event that a triggered ability looks for. Whenever the trigger event occurs, the triggered ability triggers. See rule [603](../../6-spells-abilities-and-effects/603-handling-triggered-abilities.md#rule-603), “Handling Triggered Abilities.”
