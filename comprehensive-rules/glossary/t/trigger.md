## Definition

Whenever a game event or game state matches a triggered ability’s trigger event, that ability automatically “triggers.” That means its controller puts it on the stack the next time a player would receive priority. See rule [603](../../6-spells-abilities-and-effects/603-handling-triggered-abilities.md#rule-603), “Handling Triggered Abilities.”
