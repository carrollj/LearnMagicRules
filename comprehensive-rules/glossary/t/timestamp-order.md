Source: Magic: The Gathering Comprehensive Rules glossary (effective June 19, 2026)

## Definition

A system used to determine in which order continuous effects in the same layer or sublayer are applied. See rule [613.7](../../6-spells-abilities-and-effects/613-interaction-of-continuous-effects.md#rule-613-7). See also [Dependency](../d/dependency.md).
