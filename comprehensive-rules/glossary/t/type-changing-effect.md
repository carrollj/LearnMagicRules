# Type-Changing Effect

Source: Magic: The Gathering Comprehensive Rules glossary (effective June 19, 2026)

## Definition

An effect that changes an object’s card type, subtype, and/or supertype. See rules [205.1a](../../2-parts-of-a-card/205-type-line.md#rule-205-1a)–b, 305.7, and 613.1d.
