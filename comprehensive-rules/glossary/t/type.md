Source: Magic: The Gathering Comprehensive Rules glossary (effective June 19, 2026)

## Definition

1. An object’s card type or, more broadly, its card type, subtype, and/or supertype. See rule [205](../../2-parts-of-a-card/205-type-line.md#rule-205), “Type Line,” and section [3](../../3-card-types/README.md), “Card Types.”
2. An attribute mana has. See rule [106](../../1-game-concepts/106-mana.md#rule-106), “Mana.”
