# Tiered

Source: Magic: The Gathering Comprehensive Rules glossary (effective June 19, 2026)

## Definition

A keyword ability found on some modal spells that requires you to pay an additional cost for the mode you choose. See rule [702.183](../../7-additional-rules/702-keyword-abilities.md#rule-702-183), “Tiered.”
