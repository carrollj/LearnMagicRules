# Text-Changing Effect

[Back to T entries](README.md)

Source: Magic: The Gathering Comprehensive Rules glossary (effective June 19, 2026)

## Definition

A continuous effect that changes the text that appears in an object’s text box and/or type line. See rule [612](../../6-spells-abilities-and-effects/612-text-changing-effects.md#rule-612), “Text-Changing Effects.”
