# Impending

[Back to I entries](README.md)

Source: Magic: The Gathering Comprehensive Rules glossary (effective June 19, 2026)

## Definition

An ability on some creatures that allows them to be cast for an alternative cost. If cast for its impending cost, it isn’t a creature for some period of time. See rule [702.176](../../7-additional-rules/702-keyword-abilities.md#rule-702-176), “Impending.”
