# Illegal Action

[Back to I entries](README.md)

Source: Magic: The Gathering Comprehensive Rules glossary (effective June 19, 2026)

## Definition

An action that violates the rules of the game and/or requirements or restrictions created by effects. See rule [733](../../7-additional-rules/733-handling-illegal-actions.md#rule-733), “Handling Illegal Actions.”
