Source: Magic: The Gathering Comprehensive Rules glossary (effective June 19, 2026)

## Definition

A keyword action that creates an Incubator token with a specified number of +1/+1 counters on it. See rule [701.53](../../7-additional-rules/701-keyword-actions.md#rule-701-53), “Incubate.”
