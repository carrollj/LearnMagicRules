# Remove from the Game, Removed, Removed-from-the-Game Zone (Obsolete)

[Back to glossary index](../README.md)
[Back to R entries](README.md)

Source: Magic: The Gathering Comprehensive Rules glossary (effective June 19, 2026)

## Definition

“Remove [something] from the game” is an obsolete term for “exile [something].” “The removed card” is an obsolete term for “the exiled card.” The removed-from-the-game zone is an obsolete term for the exile zone. Cards with that text have received errata in the Oracle card reference. See [Exile](../e/exile.md).
