# Roll a d20

Source: Magic: The Gathering Comprehensive Rules glossary (effective June 19, 2026)

## Definition

To roll a twenty-sided die. Similarly, a d4 is a four-sided die, a d6 is a six-sided die, and so on. See rule [706](../../7-additional-rules/706-rolling-a-die.md#rule-706), “Rolling a Die.”
