# Room

Source: Magic: The Gathering Comprehensive Rules glossary (effective June 19, 2026)

## Definition

1. A subsection of a dungeon card. See rule [309](../../3-card-types/309-dungeons.md#rule-309), “Dungeons.”
2. An enchantment subtype found on some split cards. See rule [709](../../7-additional-rules/709-split-cards.md#rule-709), “Split Cards.”
