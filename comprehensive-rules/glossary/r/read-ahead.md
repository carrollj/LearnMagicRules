Source: Magic: The Gathering Comprehensive Rules glossary (effective June 19, 2026)

## Definition

A keyword ability found on some Sagas that allows their controller to choose which chapter it starts on. See rule [702.155](../../7-additional-rules/702-keyword-abilities.md#rule-702-155), “Read Ahead.”
