# Replacement Effect

[Back to glossary index](../README.md)
[Back to R entries](README.md)

Source: Magic: The Gathering Comprehensive Rules glossary (effective June 19, 2026)

## Definition

A kind of continuous effect that watches for a particular event that would happen and completely or partially replaces that event with a different event. See rule [614](../../6-spells-abilities-and-effects/614-replacement-effects.md#rule-614), “Replacement Effects.”
