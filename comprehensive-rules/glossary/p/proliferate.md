# Proliferate

[Back to glossary index](../README.md)

Source: Magic: The Gathering Comprehensive Rules glossary (effective June 19, 2026)

## Definition

To give an additional counter to any number of players and/or permanents of each kind they already have. See rule [701.34](../../7-additional-rules/701-keyword-actions.md#rule-701-34), “Proliferate.”
