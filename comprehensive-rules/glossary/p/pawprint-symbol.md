# Pawprint Symbol

[Back to glossary index](../README.md)
[Back to P entries](README.md)

Source: Magic: The Gathering Comprehensive Rules glossary (effective June 19, 2026)

## Definition

A symbol shaped like a pawprint that is used to indicate the modes on some modal spells. See rule [700.2i](../../7-additional-rules/700-general.md#rule-700-2i).
