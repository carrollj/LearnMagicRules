## Definition

A subtype that’s correlated to the planeswalker card type. See rule [306](../../3-card-types/306-planeswalkers.md#rule-306), “Planeswalkers.” See rule [205.3j](../../2-parts-of-a-card/205-type-line.md#rule-205-3j) for the list of planeswalker types.
