## Definition

An ability which allows a prototype card to be cast with a secondary set of characteristics. See rule [718](../../7-additional-rules/718-prototype-cards.md#rule-718), “Prototype Cards,” and rule [702.160](../../7-additional-rules/702-keyword-abilities.md#rule-702-160), “Prototype.”
