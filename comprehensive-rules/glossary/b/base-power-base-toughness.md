Source: Magic: The Gathering Comprehensive Rules glossary (effective June 19, 2026)

## Definition

Effects that change the base power and/or base toughness of a creature set one or both of those values to a specific number. See rule [613](../../6-spells-abilities-and-effects/613-interaction-of-continuous-effects.md#rule-613), “Interaction of Continuous Effects.”
