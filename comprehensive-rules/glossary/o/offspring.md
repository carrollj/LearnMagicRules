Source: Magic: The Gathering Comprehensive Rules glossary (effective June 19, 2026)

## Definition

A keyword ability that allows you to pay an additional cost as you cast a creature spell to create a 1/1 token that’s a copy of that permanent when it enters the battlefield. See rule [702.175](../../7-additional-rules/702-keyword-abilities.md#rule-702-175), “Offspring.”
