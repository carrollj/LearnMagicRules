## Definition

The reference that contains the up-to-date wordings (in English) for all tournament-legal cards. A card’s Oracle text can be found using the Gatherer card database at Gatherer.Wizards.com. See rule [108.1](../../1-game-concepts/108-cards.md#rule-108-1).
