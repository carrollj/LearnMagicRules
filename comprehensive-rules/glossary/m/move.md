Source: Magic: The Gathering Comprehensive Rules glossary (effective June 19, 2026)

## Definition

To remove a counter from one object and put it on a different object. See rule [122.5](../../1-game-concepts/122-counters.md#rule-122-5).
Some older cards used “move” with respect to Auras; those cards have received errata in the Oracle card reference and now use the word “attach.”
