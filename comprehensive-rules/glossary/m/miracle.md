# Miracle

Source: Magic: The Gathering Comprehensive Rules glossary (effective June 19, 2026)

## Definition

A keyword ability that lets you cast a spell for a reduced cost if it’s the first card you draw in a turn. See rule [702.94](../../7-additional-rules/702-keyword-abilities.md#rule-702-94), “Miracle.”
