# Max Speed

[Back to glossary index](../README.md)
[Back to M entries](README.md)

Source: Magic: The Gathering Comprehensive Rules glossary (effective June 19, 2026)

## Definition

1. A player has max speed if their speed is 4. See rule [702.179](../../7-additional-rules/702-keyword-abilities.md#rule-702-179), “Start Your Engines!”
2. A keyword ability that grants an ability to the permanent or card it’s on only if that permanent’s controller (or that card’s owner, if it isn’t on the battlefield) has a speed of 4. See rule [702.178](../../7-additional-rules/702-keyword-abilities.md#rule-702-178), “Max Speed.”
