## Definition

An object with exactly one color is monocolored. Colorless objects aren’t monocolored. See rule [105](../../1-game-concepts/105-colors.md#rule-105), “Colors,” and rule [202](../../2-parts-of-a-card/202-mana-cost-and-color.md#rule-202), “Mana Cost and Color.”
