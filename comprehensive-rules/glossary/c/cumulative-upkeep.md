# Cumulative Upkeep

Source: Magic: The Gathering Comprehensive Rules glossary (effective June 19, 2026)

## Definition

A keyword ability that imposes an increasing cost to keep a permanent on the battlefield. See rule [702.24](../../7-additional-rules/702-keyword-abilities.md#rule-702-24), “Cumulative Upkeep.”
