# Commander Tax

[Back to glossary index](../README.md)
[Back to C entries](README.md)

Source: Magic: The Gathering Comprehensive Rules glossary (effective June 19, 2026)

## Definition

Informal term for the additional cost to cast a commander based on the number of times a player has cast it previously this game. See rule [903.8](../../9-casual-variants/903-commander.md#rule-903-8).
