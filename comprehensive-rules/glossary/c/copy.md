# Copy

[Back to C entries](README.md)

Source: Magic: The Gathering Comprehensive Rules glossary (effective June 19, 2026)

## Definition

1. To create a new object whose copiable values have been set to those of another object.
2. An object whose copiable values have been set to those of another object.
See rule [707](../../7-additional-rules/707-copying-objects.md#rule-707), “Copying Objects.”
