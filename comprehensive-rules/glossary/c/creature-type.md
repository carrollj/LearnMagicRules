## Definition

A subtype that’s correlated to the creature card type and the kindred card type. See rule [302](../../3-card-types/302-creatures.md#rule-302), “Creatures,” and rule [308](../../3-card-types/308-kindreds.md#rule-308), “Kindreds.” See rule [205.3m](../../2-parts-of-a-card/205-type-line.md#rule-205-3m) for the list of creature types.
