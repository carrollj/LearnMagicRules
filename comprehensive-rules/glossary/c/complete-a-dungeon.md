# Complete a Dungeon

Source: Magic: The Gathering Comprehensive Rules glossary (effective June 19, 2026)

## Definition

To remove a dungeon card from the game after reaching that dungeon card’s bottommost room. See rule [309](../../3-card-types/309-dungeons.md#rule-309), “Dungeons.”
