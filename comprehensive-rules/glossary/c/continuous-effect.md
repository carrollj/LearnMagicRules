# Continuous Effect

[Back to C entries](README.md)

Source: Magic: The Gathering Comprehensive Rules glossary (effective June 19, 2026)

## Definition

An effect that modifies characteristics of objects, modifies control of objects, or affects players or the rules of the game, for a fixed or indefinite period. See rule [611](../../6-spells-abilities-and-effects/611-continuous-effects.md#rule-611), “Continuous Effects.”
