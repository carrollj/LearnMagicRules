# Counter

[Back to C entries](README.md)

Source: Magic: The Gathering Comprehensive Rules glossary (effective June 19, 2026)

## Definition

1. To cancel a spell or ability so it doesn’t resolve and none of its effects occur. See rule [701.6](../../7-additional-rules/701-keyword-actions.md#rule-701-6), “Counter.”
2. A marker placed on an object or player that modifies its characteristics or interacts with a rule or ability. See rule [122](../../1-game-concepts/122-counters.md#rule-122), “Counters.”
