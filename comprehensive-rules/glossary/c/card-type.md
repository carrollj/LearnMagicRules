## Definition

A characteristic. Except for abilities on the stack, each object has a card type, even if that object isn’t a card. Each card type has its own rules. See rule [205](../../2-parts-of-a-card/205-type-line.md#rule-205), “Type Line,” and section [3](../../3-card-types/README.md), “Card Types.”
