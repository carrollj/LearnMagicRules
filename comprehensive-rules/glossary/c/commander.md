# Commander

[Back to C entries](README.md)

Source: Magic: The Gathering Comprehensive Rules glossary (effective June 19, 2026)

## Definition

1. A casual variant in which each deck is led by a legendary card (usually a creature). See rule [903](../../9-casual-variants/903-commander.md#rule-903), “Commander.”
2. A designation given to one legendary card in each player’s deck in the Commander casual variant.
