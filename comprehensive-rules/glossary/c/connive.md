# Connive

[Back to C entries](README.md)

Source: Magic: The Gathering Comprehensive Rules glossary (effective June 19, 2026)

## Definition

A keyword action that causes a player to draw a card, discard a card, and then to put a +1/+1 counter on a creature if a nonland card was discarded this way. See rule [701.50](../../7-additional-rules/701-keyword-actions.md#rule-701-50), “Connive.”
