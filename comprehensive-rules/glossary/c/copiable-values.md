# Copiable Values

[Back to glossary index](../README.md)
[Back to C entries](README.md)

Source: Magic: The Gathering Comprehensive Rules glossary (effective June 19, 2026)

## Definition

Values of an object’s characteristics that are checked by copy effects. See rules [613.2](../../6-spells-abilities-and-effects/613-interaction-of-continuous-effects.md#rule-613-2), [707.2](../../7-additional-rules/707-copying-objects.md#rule-707-2), and [707.3](../../7-additional-rules/707-copying-objects.md#rule-707-3).
