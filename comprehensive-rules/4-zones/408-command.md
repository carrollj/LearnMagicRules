# 408. Command

Chapter: 4. Zones
Source: Magic: The Gathering Comprehensive Rules (effective June 19, 2026)

## Rules

<a id="rule-408-1"></a>
408.1. The command zone is a game area reserved for certain specialized objects that have an overarching effect on the game, yet are not permanents and cannot be destroyed.

<a id="rule-408-2"></a>
408.2. Emblems may be created in the command zone. See rule [114](../1-game-concepts/114-emblems.md#rule-114), “Emblems.”

<a id="rule-408-3"></a>
408.3. In the Planechase, Vanguard, Commander, Archenemy, and Conspiracy Draft casual variants, nontraditional Magic cards and/or specially designated cards start the game in the command zone. Each variant has its own rules regarding such cards. See section [9](../9-casual-variants/README.md), “Casual Variants.”
