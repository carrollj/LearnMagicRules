# 806. Free-for-All Variant

Chapter: 8. Multiplayer Rules
Source: Magic: The Gathering Comprehensive Rules (effective June 19, 2026)

## Rules

<a id="rule-806-1"></a>
806.1. In Free-for-All multiplayer games, a group of players compete as individuals against each other.

<a id="rule-806-2"></a>
806.2. Any multiplayer options used are determined before play begins. The Free-for-All variant uses the following default options.

<a id="rule-806-2a"></a>
806.2a The limited range of influence option usually isn’t used in Free-for-All games. If it is, each player has the same range of influence, which is determined before play begins. See rule [801](801-limited-range-of-influence-option.md#rule-801), “Limited Range of Influence Option.”

<a id="rule-806-2b"></a>
806.2b Exactly one of the attack left, attack right, and attack multiple players options must be used. See rule [803](803-attack-left-and-attack-right-options.md#rule-803), “Attack Left and Attack Right Options,” and rule [802](802-attack-multiple-players-option.md#rule-802), “Attack Multiple Players Option.”

<a id="rule-806-2c"></a>
806.2c The deploy creatures option isn’t used in the Free-for-All variant.

<a id="rule-806-3"></a>
806.3. The players are randomly seated around the table.
