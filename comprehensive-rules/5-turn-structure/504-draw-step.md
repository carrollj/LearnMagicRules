# 504. Draw Step

Chapter: 5. Turn Structure
Source: Magic: The Gathering Comprehensive Rules (effective June 19, 2026)

## Rules

<a id="rule-504-1"></a>
504.1. First, the active player draws a card. This turn-based action doesn’t use the stack.

<a id="rule-504-2"></a>
504.2. Second, the active player gets priority. (See rule [117](../1-game-concepts/117-timing-and-priority.md#rule-117), “Timing and Priority.”)
