# 314. Schemes

Chapter: 3. Card Types
Source: Magic: The Gathering Comprehensive Rules (effective June 19, 2026)

## Rules

<a id="rule-314-1"></a>
314.1. Scheme is a card type seen only on nontraditional Magic cards. Only the Archenemy casual variant uses scheme cards. See rule [904](../9-casual-variants/904-archenemy.md#rule-904), “Archenemy.”

<a id="rule-314-2"></a>
314.2. Scheme cards remain in the command zone throughout the game, both while they’re part of a scheme deck and while they’re face up. They’re not permanents. They can’t be cast. If a scheme card would leave the command zone, it remains in the command zone.

<a id="rule-314-3"></a>
314.3. Scheme cards have no subtypes.

<a id="rule-314-4"></a>
314.4. A scheme card may have any number of static, triggered, and/or activated abilities. As long as a scheme card is face up in the command zone, its static abilities affect the game, its triggered abilities may trigger, and its activated abilities may be activated.

<a id="rule-314-5"></a>
314.5. The owner of a scheme card is the player who started the game with it in the command zone. The controller of a face-up scheme card is its owner.

<a id="rule-314-6"></a>
314.6. If a non-ongoing scheme card is face up in the command zone, and no triggered abilities of any scheme are on the stack or waiting to be put on the stack, that scheme card is turned face down and put on the bottom of its owner’s scheme deck the next time a player would receive priority. (This is a state-based action. See rule [704](../7-additional-rules/704-state-based-actions.md#rule-704).)

<a id="rule-314-7"></a>
314.7. If an ability of a scheme card includes the text “this scheme,” it means the scheme card in the command zone that’s the source of that ability. This is an exception to rule [109.2](../1-game-concepts/109-objects.md#rule-109-2).
